# 数组与矩阵

矩阵是二维数组

数组分为：空数组`A=[]`、标量`A=[5]`、向量`A=[1 2 3] A=[1,2,3](row);A=[1;2;3](col)`、二维数组`A=rand(6,6)`、高维数组`A=rand(3,4,3)`

# 向量

## 行向量

A=[向量具体值]，用空格或逗号隔开

## 列向量

A=[向量具体值]，用分号隔开或者用回车

向量转置：在向量后加一个单引号‘

A=初始值：步长：终值
//等差
